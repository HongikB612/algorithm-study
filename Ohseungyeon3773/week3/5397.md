### 연결 리스트


## 리스트 구현
구조체를 이용해 연결 리스트를 구현할 수 있다.
예시: 양방향 연결 리스트
```
typedef struct __listnode {
    char ch; 
  
    struct __listnode *next;
    struct __listnode *prev;
} Node;
``` 

각각의 노드가 이전, 다음 노드의 정보를 가지고 있다.

단일 연결 리스트의 경우 위 코드에서 prev를 삭제한다.


# 리스트 관련 계산
(그림을 그려서 파악하면 이해가 훨씬 쉽다)

- 노드를 추가하는 경우
첫 노드의 주소를 저장할 prev포인터를 생성한다. 노드를 생성할 때 next(혹은 이에 해당하는 것)를 NULL로 비워둔다. next포인터에는 다음 노드를 생성한 이후 노드의 주소를 넣어준다.

특정 노드 뒤에 새로운 노드를 추가하는 경우, 특정 노트의 next를 새로운 노드의 주소로 바꾸고 새로운 노드의 next를 특정 노드의 원래 next(가 가리키던 주소)로 바꿔준다.


- 노드를 삭제하는 경우
삭제할 노드의 next주소를 삭제할 노드의 이전 노드의 next에 넣어준다. 이후 노드를 삭제한다.
